Microsoft Web Development program obuhvata oblasti potrebne za sticanje veština za razvoj profesionalnih web aplikacija i sajtova uz rad sa bazama podataka. Pokriva sve ključne oblasti .NET tehnologije, kao i niz standardnih tehnologija potrebnih za izradu savremenih web sajtova. Polaznici će u toku programa koristiti razvojno okruženje Microsoft Visual Studio, Microsoft SQL Server, a biće obuhvaćeni i programski jezici C#, SQL, HTML i XML.
